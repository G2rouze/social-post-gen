# [App Submission] Social Media Post Generator

## Description
A beautiful, AI-powered web application that generates social media posts with custom images, captions, and hashtags using Pollinations.ai. Perfect for content creators, social media managers, and small businesses who need to quickly produce engaging social media content.

## Key Features
- 🎨 AI-Generated Images using Flux model
- ✍️ Intelligent Caption Generation using Llama 2
- #️⃣ Automatic Hashtag Suggestions
- 📱 Fully Responsive Mobile Design
- 💾 Local History Storage (up to 10 posts)
- 📋 One-Click Copy & Download
- ⚡ Fast & Lightweight (no backend required)

## Technologies Used
- **Frontend**: Next.js 15 + React 19 + TypeScript
- **Styling**: Tailwind CSS
- **AI Engine**: Pollinations.ai API
  - Image Generation: Flux model
  - Text Generation: Llama 2 7B
- **Deployment**: Vercel-ready

## How It Works
1. User enters a description of what they want their social media post to be about
2. The app simultaneously generates:
   - A professional, eye-catching image via Flux
   - An engaging, platform-optimized caption via Llama 2
   - Relevant and trending hashtags via Llama 2
3. User can copy any part individually or download the image
4. All posts are saved locally for quick access
